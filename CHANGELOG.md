## Release Notes

### Version 1.0.0 - Initial Release

This is the first stable release of the Community Event Template. It includes everything you need to create a
professional event website for your tech community.

**What's included:**

- Complete, production-ready template
- Fully responsive design
- Easy customization through content files
- Comprehensive documentation
- Modern tech stack
- Open source (MIT License)

**Perfect for:**

- Tech conferences
- Community meetups
- Developer events
- Workshops & hackathons
- Any community gathering

**Quick Start:**

```bash
git clone https://github.com/sunnat629/community-event-template.git
cd community-event-template
npm install
npm run dev
```

**Customization:**
All content is in `src/content/` - update these files to customize your event:

- `hero.ts` - Hero section
- `eventDetails.ts` - Event details & venue
- `speakers.ts` - Speaker information
- `partners.ts` - Sponsors & partners
- `organizers.ts` - Organizer information
- `agenda.ts` - Event schedule

See [README.md](README.md) for full documentation.
